Kamera Inventar Pro — Einrichtung

Einmalige 5-Minuten Einrichtung für Login & Cloud-Speicherung

1

Firebase Projekt erstellen

Gehe zu console.firebase.google.com → Klicke „Projekt hinzufügen" → Name eingeben (z.B. „kamera-inventar") → Weiter bis Projekt erstellt ist.

2

Web-App hinzufügen & Konfiguration kopieren

Im Firebase-Projekt: Klicke das </> Symbol → App registrieren → Gib einen Namen ein → Die angezeigte Konfiguration brauchst du für Schritt 5.

3

Authentication aktivieren

Im Firebase-Menü: Authentication → Los geht's → Google → Aktivieren → Projekt-Support-E-Mail wählen → Speichern.
Dann unter Authentication → Settings → Authorized Domains hinzufügen: jonasw1404-arch.github.io

4

Firestore & Storage aktivieren

Firestore Database → Datenbank erstellen → Im Produktionsmodus starten → Standort wählen (europe-west1) → Aktivieren.
Dann unter Regeln diese Regeln einfügen:

rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /users/{userId}/{document=**} { allow read, write: if request.auth != null && request.auth.uid == userId; } } }

Storage → Los geht's → Im Produktionsmodus starten → Aktivieren.
Dann unter Regeln:

rules_version = '2'; service firebase.storage { match /b/{bucket}/o { match /{allPaths=**} { allow read, write: if request.auth != null; } } }
5

Firebase Konfiguration eingeben

Kopiere die Werte aus deiner Firebase Web-App-Konfiguration hierhin:
